버드와이저 - 스탤라 광고

서버 구성리눅스 서버 [ AWS EC2 ]
언어PHP 7.2
프레임워크코드이그나이트3
데이터베이스MYSQL [ RDS ]
프론트엔드HTML, CSS 디자인팀에서 전달받았습니다.
백엔드PHP와 jquery 를 사용하여 소통하며 혼자 개발하였습니다

버드와이저 QR 이벤트 페이지 - 뉴욕

어드민페이지 소개:

  • 로그인, 엑셀 다운로드, 통계 페이지 포함
  • AWS EC2에 서버 설정 후, Route53으로 도메인 등록
  • SSL은 Certificate Manager를 이용한 설정
  • 로드 밸런서를 통한 서버 운영

이벤트 페이지 소개:

  • 성인 대상 QR 이벤트 페이지
  • 본인인증을 통한 참여 가능
  • 체크플러스를 활용한 본인인증
  • 내장된 GA(Google Analytics) 태깅 코드

기타 정보:
