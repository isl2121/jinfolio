랜덤투유

랜덤투유 바로가기

랜덤투유는 ADMIN 페이지를 담당하여 전면 리뉴얼한 프로젝트입니다. 결제 모듈은 다른 개발자에게 서포트를 받아 작업하였고, API 부분은 외부 개발자가 개발했습니다. 기존에 오래된 코드로 유지보수가 어려웠기 때문에 처음부터 끝까지 재작업하였습니다.

개발 내용

  • API와의 데이터베이스 연동을 통한 리스트 관리 페이지 구현
  • 어드민 페이지 전반적인 기능 담당
  • AJAX를 사용한 필요 기능 개발 (예: 상품 등록 시 카테고리 불러오기)
  • 엑셀 업로드 기능 추가 및 다운로드 기능 구현 (예: 이벤트 당첨자에게 포인트 지급)

랜덤투유는 언어 교체가 가능하도록 개발되어 있어 해외 진출을 고려한 플랫폼입니다. 코드이그나이트 안에있는 내용을 활용하여 언어 선택이 가능하도록 되어있습니다.
